# Monash Robotics Wiki

Lab wiki for students and researchers at Monash Robotics.

**Live site:** https://monashrobotics.github.io/wiki/

## Contents

- **Onboarding** — getting started, new PhD checklist, HPC access guide
- **HPC** — M3 (Monash), Pawsey Setonix, NCI Gadi
- **Storage** — S Drive (shared lab storage)
- **Presentations** — skill-sharing and paper reading slides

Anyone in the lab can contribute — see the [contributing guide](https://monashrobotics.github.io/wiki/contributing/) for options including quick GitHub edits, local editing, and AI-assisted drafting.
