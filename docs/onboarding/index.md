# Getting Started

Welcome to Monash Robotics. Here's where to begin.

1. **Join Slack** — [monashrobotics.slack.com](https://monashrobotics.slack.com). Ask your supervisor for an invite.
2. **Complete lab induction** — required before you can access the lab. See [Lab Induction](lab-induction.md).
3. **Get HPC access** — most lab work requires M3, Pawsey, or NCI. See [HPC Access](hpc-access.md).
4. **Read your role-specific checklist:**
   - [New PhD Student Checklist](phd.md)
