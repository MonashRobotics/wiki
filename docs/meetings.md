# Lab Meetings

Two recurring weekly meetings — both held Thursdays at the same location (18 Alliance Ln, Clayton, G-G15-Lounge, MechEng) with Zoom for remote attendance.

To be added to either meeting, contact Dana Kulić (`dana.kulic@monash.edu`) to request a calendar invite.

---

## Monash Robotics Weekly Meeting

| | |
|---|---|
| **When** | Thursdays 10:00–11:00 AM AEST |
| **Where** | 18 Alliance Ln, Clayton — G-G15-Lounge (MechEng) + Zoom |
| **Organiser** | Dana Kulić |
| **Audience** | All Monash Robotics lab members |

The weekly lab meeting covers two types of presentations:

- **Skill-sharing** — tutorials on tools, workflows, HPC, software (anyone can present)
- **Research updates** — progress presentations on ongoing projects (unpublished)

Slides from skill-sharing sessions are archived publicly. Research slides are kept in a private repo for org members only.
